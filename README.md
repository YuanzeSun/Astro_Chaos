# 天文闹赛 Astro Chaos

《天文闹赛》是一个本地网页端社团养成游戏。玩家扮演高中天文社指导老师，在三年内管理社员、经费、设备、训练方向和比赛节奏，目标是把学生送进 IOAA 并拿到奖牌。

游戏可选 OpenAI-compatible API 接入。AI 可以根据当前局势生成锐评，并偶发生成自动生效的被动事件，或加入本周行动列表的可选事件。

## 游戏玩法

- 每周选择一个社团活动，推进训练、恢复压力、维修设备、招募新社员或争取赞助。
- 四项能力决定比赛表现：理论、观测、实测、常识。
- 压力过高可能导致退社，士气和设备会影响训练稳定性。
- 天气会影响观测类活动，部分学生特性也会改变天气、成长或比赛波动。
- 赛事按时间推进：市级预赛、省级复赛、CNAO 国初、CNAO 国决、IOAA。
- 训练、恢复、设备维护、招新和赞助会互相争夺每周行动机会。

## 运行方式

```bash
python3 main.py
```

浏览器打开：

```text
http://127.0.0.1:8765/
```

指定端口：

```bash
python3 main.py --port 9000
```

## AI 锐评与事件

启动游戏后，在网页右上角点击 `API 设置`，填写：

- `API Token`
- `Base URL`，可为空；仅使用自定义网关时填写，例如 `https://your-openai-compatible.example.com`
- `模型`，点击“获取模型”后从服务器返回的可用模型中选择
- `Wire API`，默认是 `Responses`
- `AI 模式`，可选择“锐评 + 随机事件”或“仅锐评”


## 文件结构

```text
main.py              # 本地网页服务与 AI API 代理
data.py              # 旧版导入兼容
web/index.html       # 页面结构
web/styles.css       # 页面样式
web/app.js           # UI 交互、设置、AI 事件操作
web/gameEngine.js    # 游戏状态、数值、赛事和事件逻辑
web/gameData.js      # 学生特性、训练、赛事、随机事件等配置
```

## 说明

本游戏完全虚构，题材参考天文竞赛社团生活，但不对应真实赛事规则。
