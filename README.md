# 🌌 天文闹赛 (Astronomy Chaos Competition)

这是一个基于 **Python** 的命令行模拟养成游戏。玩家将扮演一名天文社团的指导老师，在三年内培养一批高中生社员，带领他们参加从市赛到国际奥林匹克天文与天体物理竞赛（IOAA）的各项比赛。

---

## 🎬 背景

高二那年，你最后一次参加天文闹赛选拔赛，但遗憾未能进入集训队。

一觉醒来，你坐在天象厅里，发现自己成为了一所高中的天文社指导教师。

回想起当年的遗憾，你决定要改变这一切。

> **声明**：代码主要由 AI 生成，游戏完全虚构，与现实中的天文竞赛无直接对应。

---

## 🚀 项目简介

游戏目标是在有限的时间（三年）与资金限制下，通过合理训练与压力管理，培养出具备优秀 **理论、观测、实测、常识** 能力的顶尖选手，并最终赢得 IOAA 国际赛。

---

## ✨ 核心机制

### **属性养成**
学生拥有四个主要属性：

- 理论
- 观测
- 实测
- 常识

这些属性通过不同训练方式提升。

### **压力管理**
训练、比赛及事件会增加压力；过高压力可能导致退社。

### **天赋系统**
每位学生都拥有随机天赋（如“阴天教”、“Furry”、“理论天才”等），影响他们的学习效率或压力敏感度。

### **天气影响**
观测类训练及部分比赛会受到天气变化影响。

### **比赛晋级关系**

- 市赛晋级 → 省赛  
- 国初 → 国决 → IOAA  
- **国初参赛不依赖省赛**

## 💻 如何运行

本项目仅依赖：

- Python 标准库
- `colorama`（可选，仅用于彩色输出）

### **安装依赖（推荐）**
```bash
pip install colorama
```

运行游戏:
```bash
python main.py
```
## 🎮 游戏玩法

游戏以周为单位推进。每周末，你需要：

- 检查社员状态： 关注学生的属性等级和压力值。

- 管理资金： 资金透支会带来负面影响。

- 选择训练计划： 从随机生成的训练选项中，选择本周执行的活动。训练会提升属性、调整压力，并消耗资金。

- 应对随机事件： 随机事件可能带来资金、压力或属性的意外变动。
